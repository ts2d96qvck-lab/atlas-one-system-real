# Guia Rápido — Atendente

Referência de bolso para o dia a dia no Atlas One.

---

## Entrar
**http://app.atlasone.local.gd** → ID empresa + e-mail + senha

---

## Atender

```
1. Abrir conversa na lista
2. Ler histórico
3. Responder (Enter envia / Shift+Enter quebra linha)
4. Fechar quando resolver
```

---

## Atalhos úteis

| Ação | Como |
|------|------|
| Responder mensagem específica | Duplo clique na mensagem |
| Enviar arquivo | Clipe 📎 → preview → Enviar |
| Gravar áudio | Microfone 🎤 → Enviar áudio |
| Resposta rápida | Hashtag `#tag` no Admin |
| Transferir | Topo → atendente → Transferir |
| Editar nome cliente | Topo → lápis ✏️ |

---

## Prioridades

🔵 **Bolinha azul** = mensagem nova  
🔴 **+5m** = cliente esperando → responder primeiro

---

## Status

- **Aberto** = atendendo agora
- **Aguardando** = esperando cliente
- **Fechar** = terminou (some da sua fila)

---
