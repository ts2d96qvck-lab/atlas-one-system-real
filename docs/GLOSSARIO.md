# Glossário — Atlas One

| Termo | Significado |
|-------|-------------|
| **Tenant** | Empresa/cliente isolado no sistema (multi-tenant) |
| **Slug** | ID único da empresa usado no login (ex.: `atlas-one`) |
| **Dono (owner)** | Proprietário da conta com acesso total |
| **Inbox** | Central de conversas WhatsApp |
| **Lead** | Oportunidade comercial no CRM |
| **Pipeline / Funil** | Etapas do processo de venda (Kanban) |
| **Departamento (Team)** | Área da equipe (Comercial, Novos, Suporte) |
| **Instância WhatsApp** | Número conectado via Evolution API |
| **Evolution API** | Servidor que conecta ao WhatsApp Business/API |
| **Webhook** | Notificação automática de mensagens recebidas |
| **2FA** | Autenticação em duas etapas via SMS |
| **Atalho / Hashtag** | Resposta rápida predefinida (ex.: `#boasvindas`) |
| **Automação** | Regra que executa ação ao ocorrer gatilho |
| **Gatilho (trigger)** | Evento que dispara automação (ex.: lead mudou etapa) |
| **Transferência** | Mover conversa para outro atendente |
| **Fila Novos** | Departamento de entrada de clientes inbound |
| **SLA +5m** | Alerta quando cliente aguarda resposta > 5 minutos |
| **Audit log** | Registro de ações administrativas |
| **Proxy nginx** | Servidor que unifica URL web + API na porta 80 |
| **Glass panel** | Estilo visual translúcido da interface |
